对于Service和DAO类，基于SOA的理念，暴露出来的服务一定是接口，内部的实现类用Impl的后缀与接口区别

Examples:

    public interface DemoService{

        void f();

    }



    public class DemoServiceImpl implements DemoService {

        @Override

        public void f(){

            System.out.println("hello world");

        }

    }
